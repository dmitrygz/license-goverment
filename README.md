# License Goverment

Минималистичное веб-приложение для учета лицензий с авторизацией, ролями пользователей, журналом действий, историей операций и админ-панелью.

## Обзор

`License Goverment` помогает быстро учитывать лицензии по сменам `День` и `Ночь`, считать суммы, хранить историю действий и разграничивать права между обычными пользователями и администраторами.

## Что умеет

- регистрация и вход в аккаунт
- роли `admin` и `user`
- добавление лицензий кнопками `Сдал` и `Не сдал`
- учет по сменам `День` и `Ночь`
- автоматический расчет:
  - общей суммы
  - суммы в казну
  - суммы на руки
- история операций по датам
- журнал действий пользователей
- управление ролями через админ-панель
- удаление аккаунтов администратором
- светлая и темная темы
- автосохранение рабочей сессии

## Интерфейс

Приложение разделено на несколько вкладок:

- `Работа`
  Основной экран для учета лицензий и удаления выбранных записей.

- `Настройки`
  Переключение темы и изменение тарифов.

- `История`
  Просмотр прошлых оформлений по диапазону дат.

- `Журнал`
  История действий пользователей в системе.

- `Админы`
  Управление ролями и аккаунтами.

## Роли пользователей

### `user`

- может входить в систему
- может добавлять и удалять лицензии
- не может видеть журнал и историю
- не может редактировать тарифы

### `admin`

- имеет доступ к журналу и истории
- может изменять тарифы
- может назначать администраторов
- может понижать администраторов до `user`
- может удалять аккаунты

Первый зарегистрированный пользователь автоматически становится администратором.

## Хранение данных

Приложение поддерживает два режима хранения:

- локально: `SQLite`
- в облаке: `PostgreSQL` через `DATABASE_URL`

## Безопасность

- пароли не хранятся в открытом виде
- используется SHA-256 хэш
- действия пользователей логируются
- доступ к админским разделам ограничен ролями
